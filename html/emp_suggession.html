Suggestion Box


Welcome to CeltPayroll. Using this page you can give your suggests.

  • Enter the following details :
    • Select " Type ".
    • Select " Rep.Manager/HR " and You get the list of Manager/HR.
    • Select " Manager/HR name ".
    • Enter " Subject "
    • Enter " Description "
    • Enter " Response "
  • Click " BROWSE " Button to add files or images for suggest .
  • Click " SAVE " Button to save the suggest.
  • Click " SUBMIT " to submit the suggest.
  • Click " CLOSE " Button to close window.
  • Click " -> " arrow to Edit the suggest.
  • Click " UPDATE " Button to save the updated suggestion.
