Attendance History


Welcome to CeltPayroll. Using this page you can enter Attendance History user manually.


Attendance History User Manually:-

    From here you will be get the Attendances information of any employee in the organization for any month.

  • Admin Attendance History:

      • Here you will be get information of any admin attendance for selected Period.
    • Enter "From Date"(Start of Attendance Report)
      •
    • Enter "To Date"(End of Attendance Report)
      •
    • Select "Department"
      •
    • Select "Grade"
      •
    • Select "Employee Name"
      •
    • Click On button to Submit Admin information to Attendance History.
      •
    • Click On button to get Attendance History excel Sheet.
      •
    • Click On button to Close Attendance History.
      •

  • Employee Attendance History:

      • Here you will be get information of any Employee attendance for selected Month.
    • Select "Unit"(Branch Name)
      •
    • Select "Month"
      •
    • Select "Year"
      •
    • Click On button to Submit Admin information to Attendance History.
      •
    • Click On button to Close Attendance History.
      •
