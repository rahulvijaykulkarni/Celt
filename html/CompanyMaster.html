Company Master User Manually:

    Using Company Master you can stored all information About the Company. Related to the Registration Details of Varies Agencies of government.

  • Enter the following details for Company Master:

    • "Company code"(It will be unique and system generated)
      •
    • Enter "Company Name"(Here entered the Registered Name of Company)
      •
    • Enter "Company Address"(Here entered the Registered Address of Company)
      •
    • Enter "Employee Series"(Here entered the Series of Employee ID Ex.I00)
      •
    • Select "State"(The State in Which make Registered Company)
      •
    • Select "City"(The City in Which make Registered Company)
      •
    • Enter "Pin code"(Pin code of Restored Address Area)
      •
    • Enter "Admin User Id"(Here Entered Admin User Id of Company)
      •
    • Enter "Admin Password"(Here Entered Admin Password of Company)
      •
    • Enter "File No"(Which is Unique of all Company)
      •
    • Select "PayPro No"(Enable or Disable)
      •
    • Enter "Company Logo" by Click on button(document must in only JPG or PNG Format)"
      •
    • This will make Use for the billing and send legal documents.
      •
