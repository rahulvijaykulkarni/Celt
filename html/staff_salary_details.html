Welcome to CeltPayroll. Using this page you can enter Staff Employee Salary manually.


Staff Salary Structure User Manually:-

    Using this we calculated the staff salary according to the salary structure.

  • Enter the following Staff Staff Salary Structure details:

    • Select "Client Name"(It must be always Staff)
      •
    • Select "Unit Name"
      •

  • Employee Salary Details:

    • Select "Staff Name"
      •
    • Select "Job Type"
      •
    • Select "Status"(Salary Calculation Monty/Daily)
      •
    • Enter "CTC(Annual Income)"
      •

  • Earning Heads:

      • Here enterd the detail's of following earnig heads.
    • Enter "Basic"
      •
    • Enter "HRA"
      •
    • Enter "Conveyance"
      •
    • Enter "MIS Allowance"
      •
    • Enter "MEDI Allowance"
      •
    • Enter "SPEC Allowance"
      •
    • Get "Total"(Of Earning Heads)
      •

  • Lumpsum Heads:

      • Here enter the information about Lumpsum head of selected Staff.

  • Deduction Heads:

      • Here enter the information about Deduction head of selected Staff.

  • Tax Deductions:

      • Here enter the information about Tax Deductions for selected Staff.
    • Select "PF Employee"(Applicable Yes/No)
      •
    • Select "ESIC Employee"(Applicable Yes/No)
      •
    • Select "PF Employer"(Applicable Yes/No)
      •
    • Select "ESIC Employer"(Applicable Yes/No)
      •
    • Select "LWF"(Applicable Yes/No)
      •
    • If LWF Applicable then Select "LWF Actual/Monthly"
      •
