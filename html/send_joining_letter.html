Employee Joining Letter/Client Feedback Email


Welcome to CeltPayroll. Using this page you can enter Employee Joining Letter manually.


Employee Joining Letter User Manually:

    Using this you will make send the joining letter to the employee, as well as Send feedback link to Customer and also send the Reliving letter to the employee.

  • Enter the following Employee Joining Letter details:

    • Select “Client Name”
      •
    • Select “State Name”
      •
    • Select “Branch Name”
      •

    • Joining Letter:

    • Using this you will be make send joining the letter to the employee in the patriculer barnch.
      • Click On Button For sending the Joining Letter the Employee.
        •
      • Click On to getting Details of Joining Letter this view in the grid view of the same page.
        •

    • Feedback Letter:

    • Using this you wil be make send the Feedback link to employee in the particuler branch and also get the Report of this.
      • Select “Month and Year”
        •
      • Click On Button to Send the Feedback link to particuler Branch Employee.
        •
      • If you want Feedback Report then Click On Button to get Report of Feedback.
        •

    • Relieving Letter:

    • Using this you wil be make send the Relieving Letter to employee in the particuler branch and also get the Report of this.
      • Select “Left Employee”
        •
      • Click on Button to Send Relieving Report to Employee.
        •
