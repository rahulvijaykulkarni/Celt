Return Material


Welcome to CeltPayroll. Using this page you can enter Return Material user manually.


Return Material User Manually:-

    Here you will be get the information about material which send to client and all information releted to this like Dispatch of inventory and Courier.

  • Return Material:

      • Here you will be get the information about Return material which come from client.
    • Select “Client Name”
      •
    • Select “State Name”
      •
    • Select “Branch Name”
      •
    • Select “Invoice No”
      •
    • Enter “Return Date”
      •
    • Enter “Return Accepted By” (Name of the person how Accept return Material)
      •
    • Enter “Reason” (Reason of Material Return)
      •

  • Dispatch Bill:

      • Here you will be create the Dispatch bill of material for selected client.
    • Select “Receiver Type” (Material, Invoice, Duplicate Id Card)
      •
    • Select “Dispatch By” (By Courier, By Postal, By Hand)
      •
    • Select “Client Name”
      •
    • Select “State Name”
      •
    • Select “Branch Name”
      •
    • Select "Employee Name"
      •
    • Enter “Dispatch Date”
      •
    • Click On Button to Upload Receiving Report in System.
      •
    • For Search Return Material detail’s Client wise Select "Client Name"
      •
    • Use Search Box for Search Employee from Grid View
      •
    • If you want to edit any detail’s then click on Button in Grid view.
      •

  • Dispatch Inventory:

      • Here you will be get the informatin about Dispatch fo inventory against POD No.
    • Enter “Employee Name”
      •
    • Enter “Document Type”
      •
    • Enter “Dispatch Date”
      •
    • Enter “Receiving Date”
      •
    • Enter “POD No”
      •
    • Enter “Shipping Address”
      •
    • Click On Button to Upload Receiving Report in System.
      •
    • Click On Button to save Dispatch Details of Inventory.
      •

  • Send Email:

      • Using this make send mail of Dispatch of Material and get report of dispatch material.
    • Select “Client Name”
      •
    • Select “State Name”
      •
    • Select “Branch Name”
      •
    • Click On Button to send Return Material Detail’s Mail to Client.
      •
    • Click On Button to Get Send Email Report.
      •

  • Courier:

      • Using this you will be make save information about the courier and track the courier.
    • Select “Category” (of Material)
      •
    • Enter “Date”
      •
    • Select “Client Name”
      •
    • Select “State Name”
      •
    • Select “Branch Name”
      •
    • Enter “Address”
      •
    • Enter “Weight” of Courier
      •
    • Select “Unit” of Material (Kg./Gram.)
      •
    • Enter “No of Packet”
      •
    • Enter “POD No”
      •
    • Select "Delivery Status"(Y/N)
    • Click On Button to Save Courier Detail’s.
      •
    • Click On Button to Update Courier Detail’s.
      •
    • Click On Button to Delete Courier Detail’s.
      •
    • Click On Button to Track the Courier location.
      •
