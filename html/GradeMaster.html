Designation Master


Welcome to CeltPayroll. Using this page you can enter Designation Master manually.


Designation Master User Manually:-

    In the Designation Master Make store the information about which employee will be make work on which designation in the organization.
  • Select "Employee Type"(Permanent, Staff, Temporary, Repair & Maintenance)
    •
  • Enter "Designation Code"
    •
  • Enter "Designation Description"
    •
  • Select "Reporting To"
    •
  • Select "Categories" (of Employee Skilled, Semi-Skilled, Un-Skilled)
    •

  • Click On button to Save Designation Master.
    •
  • Click On button to Clear data which entered in Designation Master before save.
    •
  • Use the Search Tab for Search the Particular City or State from the Grid View for making update information of Designation Master.
    •
  • Click On button to Update Designation Master.
    •
  • Click On button to Delete From Designation Master.
    •
  • Click On button to get records in excel which present in Grid view.
    •
  • Click On button to get Print of records in Grid View.
    •
  • Click On button to get Copy of records in Grid View for Past in any were.
    •
  • Click On button to select which colume will make disply in Grid View.
    •
  • Click On button to Close Designation Master.
