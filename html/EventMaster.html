Event Master


Welcome to CeltPayroll. Using this page you can enter Event Master details .


  • Enter the following Event Master details:
    • Enter " Header ".
    • Enter " Message ".
    • Enter " Start Date ".
    • Enter " End Date ".
    • Click " Browser " to add " Event File ".
  • Click " SAVE " Button to Save and Display the event in table.
  • Click " CLEAR " to Clear the event.
  • Click " EXPORT TO EXCEL " Button to see execl sheet.
  • Click " CLOSE " Button to close the form.
  • Click " Table " to display information in all the Fields.
