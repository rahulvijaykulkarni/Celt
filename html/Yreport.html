Yearly Report


Welcome to CeltPayroll. Using this page you can See Yearly Reports .

  • Select " Select Unit "
  • Select " Month " from calender button For example(mm/yyyy).
  • Select " Year " from calender button For example(mm/yyyy).
  • Select " Emp Code "
  • Click " SHOW " button to display monthly reports in Enquiry Form.
  • Click " CLOSE " button to Close Enquiry Form.
  • Click " SEND EMAIL " button to send email to valid id in Enquiry Form.
