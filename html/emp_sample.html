Employee Attendance


Welcome to CeltPayroll. Using this page you can enter Employee Attendance manually.


Employee Attendance User Manually:-

    Using this we will be make upload the attendance of employee as well as upload the material and covency of detail’s of employee for making the there bill and Payment.

  • Employee Attendance:

      • Using this we will make entered the attendance of employee for particular month. for make process bill and payment.

    • Select “Month/Year”
      •
    • Select “Client Name”
      •
    • Select “State Name”
      •
    • Select “Branch Name”
      •
    • Click On Button for Make process the Attendance for billing.
      •
    • Select "Employee attendance from Grid view by tick on check box one by one"
      •
    • If any Employee will be left in the particular month then enter the left date and enter the Reason of left.
      •
    • Click On Button for submit the Attendance in the system.
    • Make Fill the Attendance of all Employee for full month by using P= Present, A=Absent, W=Week off.
      •
    • Click On Button for save the attendance which entered in the system.
      •
    • Then write the Description and select the document of attendance for upload in the system.(Only in JPG,PNG,JPEG and PDF format)
      •
    • Click On Button for Upload the Attendance document in the System.
      •
    • Then Click On Button< for make Approved the Attendance
      •
    • IF you want to report of Attendance then click on Button to Get the Attendance Report.
      •
    • IF you want to Update the Attendance Report then upload the File by click on Button to insert Attendance Document in System.
      •

          • Using this we will make entered the Material information for make process bill and payment of Material.

        • Select “Month/Year”
          •

          •

          •
