Service Request User


Welcome to CeltPayroll. Using this page you can enter Service Request User manually.


Service Request User Manually:-

    Here we get the Client Feedback according to the Zone, Region and Branch. Get Request of the Client for New Deployment. Get Request of Supervisor for Particular Equipment form working, Information about Raised Complains which entered by supervisor for making action.

  • Feedback:

      • Here we get the Feedback which come From client as Zone wise, Region wise and Branch wise for make Improvement in our Services.

    • Select “Client Name”
      •
    • Select “Zone”
      •
    • Select “Zone User Name”
      •
    • Select “Region of User Name”
      •
    • Select “Brach Feedback”
      •

  • Client Request:

      • Here we get the information about the Request of client for new Deployment in the Particular branch for Particular Designation.

    • Select “Client Name”
      •
    • Select “Branch Name”
      •
    • Select “Service Type”
      •
    • Enter “Comment”
      •
    • Enter “Request Date”
      •
    • Select “Status”
      •
    • Click on Button to upload the Client Request Document in System.
      •
    • Click on Button to Save/Submit Request in/For System.
      •
    • Click on Button to Cleat data which insert in the Client Request befor Save.
      •
    • Click on Button to Close Service Request.
      •

  • Supervisor Request:

      • Using this Supervisor will be fill there Requirements in the System with the Quotations of this Material which need for make work on the Filed.

    • Select “Client Name”
      •
    • Select “State Name”
      •
    • Select “Branch Name”
      •
    • Select “Service Type”
      •
    • Select “Vendor Type”
      •
    • Select “Vendor Name”
      •
    • Enter “Product Specification”
      •
    • Enter “Type/Make”(Type of product or how to make this product)
      •
    • Enter “Mobile No”
      •
    • Enter “Costing/Amount” (Enter the Product Price)
    • Click on Button to upload Quotation in System.
      •
    • Click on Button to save this Quotation in the System.
      •
    • Click on Button to Submit this Quotation to Finance Department.
      •
    • Click on Button to Clear data Before save in the service Request.
      •
    • Click on Button to Close Service Request.
      •
