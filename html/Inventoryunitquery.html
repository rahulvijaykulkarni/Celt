Stock Status Unitwise


Welcome to CeltPayroll. Using this page you can enter Head Type Master information.

  • Enter the following Head Type Master details:
    • Select " Unitwise " button.
    • Enter " Head Type Name ".
    • Select " Head Type ".
    • Select " Active Status ".
  • Click " PROCESS " Button to process the information.
  • Click " CLEAR " Button to clear the Stock Status unitwise.
  • Click " UNIT CODE " in table to see " Unitwise " Details.
  • Click " CLOSE " Button to close the Stock Status Unitwise form.
