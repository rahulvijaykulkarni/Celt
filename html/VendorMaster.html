Vendor Master


Welcome to CeltPayroll. Using this page you can enter Vendor Master manually.


Vendor Master User Manually:-

    In the Vendor Master we enter the all information about the vendor like Address, Contact, Service Category, GST, SAC Code, Bank, etc,.

  • Vendor Info:

      • Here we entered the information about vendor with Vendor Category and Agreement Start Date/End Date.
    • Enter “Vendor ID"(System Generated)
    • Enter “Vendor Name"
      •
    • Select “Vendor Type"(As per Nation Indian/International)
      •
    • Select “Vendor Type"(Private / Public)
      •
    • Enter “Vendor Address”
      •
    • Enter "Contact Person 1 Name”
      •
    • Enter "Contact Person Mob No.1”
      •
    • Enter "Contact Person 2 Name”
      •
    • Enter "Contact Person Mob No.2”
      •
    • Enter "Contact Person Email ID”
      •
    • Select “Vendor Categories”
      •
    • Enter “Credit Period”(IN Day)
      •
    • Enter "Area to be Served"
      •
    • Enter "Agreement Start Date"
      •
    • Enter "Agreement End Date"
      •

  • All Taxes:

      • Here we entered the Detail’s Related to Taxes of the Vendor like GST No., PAN No., Services Type, SAC Code.
    • Enter “GST IN” (Enter the Vendor GST No)
      •
    • Enter “PAN No” (Enter the Vendor PAN No)
      •
    • Enter “Service Type” (Enter the Vendor Service Type)
      •
    • Select “Active Status” (Active/Deactivate)
      •
    • Enter “Total Dues”
      •
    • Enter “Opening Balance”
      •
    • Enter “Registration No”
      •
    • Enter “SAC Code” (Service Accounting Code)
      •

    • Enter “Attention”

    • Enter “Vendor Bank IFSC Code”

  • Contact Details:

      • Here we entered the Contact Person Details of Vendor as per there Department and Designation.

  • Weight Master:

      • Here we make assign Quotation of Courier according to the Weight and Kilometer(KM).
    • Select “Courier Type”(By Road, By Train, By Air)
      •
    • Select “Weight”(IN Gram, KG)
      •
    • Select "From Weight"
      •
    • Select "To Weight"

      •
    • Select "To KM"
      •
    • Enter "Rate"
      •  Click on button to Add Weight Detail's in Vendor master.

  • Rate Master:

      • Here we make Add the Rate of Particular Item of this Vendor.

    • Enter "Item Size"
