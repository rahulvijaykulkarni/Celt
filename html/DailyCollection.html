Daily Collection


Welcome to CeltPayroll. Using this page you can enter Daily Collection details .

  • Enter the following Daily Collection details:
    • Select " Type ".
    • Select " Month ".
    • Select " Full Month " or " Daily ".
    • Select " Year ".
    • Select " Date " from calendar button For example(dd/mm/yyyy).
  • Click " PROCESS " Button to display Daily Collection.
  • Click " EXPORT TO EXCEL " to see excel sheet.
  • Click " CLOSE " Button to close the Daily Collection form.
  • Click " OPENING UPDATE " to see available updates
