Telescopic Gate

Welcome to Avians. Using this page you can enter values manually.

Client Details
  • Enter the following client details:
    • Enter " Client Name ".
    • Select " Client Location ".
    • Select " Project Location ".
    • Select " Distance ".
    • Select " Product Installation Category ".
    • Enter " Project Name ".
    • Enter " Project Contact Person Name 1 ".
    • Enter " Project Contact Person No.1 ".
    • Enter " Project Contact Person Email Id 1 ".
    • Enter " Project Contact Person Name 2 ".
    • Enter " Project Contact Person No.2 ".
    • Enter " Project Contact Person Email Id 2 ".
    • Enter " Architect/Consultant Name ".
    • Enter " Architect/Consultant Contact Person No. ".
    • Enter " Architect/Consultant Contact Person Email Id ".
Technical Specifications
  • Enter the following technical details:
    • Enter " Width in meters ".
    • Enter " Height in meters ".
    • Enter " Total No.Of Leaf ".
    • Enter " Quantity ".
    • Select " Operation Type ".
    • Select " Leaf Type ".
    • Select " Material ".
    • Select " Outer Frame Mmebers " and Accordingly " Internal Members " will get generated.
    • Select " subject ".
    • Select " Sheeting Details - MS/GI ", Accordingly Select " MS/GI Type "," Thickness " and also Specify " Width " and " Height ".
    • Select " Sheeting Details - Acrylic / Polycarbonte ", Accordingly Select " Type "," Thickness " and also Specify " Width " and " Height ".
    • Select " Pretreatment Process ".
    • Select " Material Finish ".
    • Enter " RAL Code ".
    • Select " Wicket Gate " If ' Required ' then Enter " Sizes ".
    • Select " Optional Add Ons " .
Advance Control Accessories
  • Enter the following Advance Control Accessories:
    • Operating Accessories:
      • Enter " Extra Push Button ".
      • Enter " Remote Control With Receiver".
      • Enter " Extra Remote Control ".
      • Enter " Magnetic Loop Detector ".
    • Safety Accessories:
      • Enter " Safety Photocell ".
      • Select " Vertical Safety Edge Sensor ".
      • Select " Vertical rubber Safety Edge ".
    • Indicative Accessories:
      • Enter " Flashing Lamp ".
      • Enter " Flashing Lamp with Alarm ".
      • Enter " Traffic Lamp ".
    • Mechanical Accessories:
      • Enter " Fixed Bollards ".
Additional Accessories (click on " Add More " Button)
  • If you want to Add New Accessories
    • Enter " Description ".
    • Enter " Size ".
    • Enter " Quantity ".
    • Select whether to "Add to Product " or "Add to Proposal ".
Additional Requirement
  • Select the following Additional Requirement :
    • Select " Extended Warranty ".
    • Select " Interlocking " Accordingly Enter " Quantity ".
    • Select " Utility Cost ".
    • Select " Taxes On Supply ".
    • Select " Taxes On Installation ".
  • Click " Submit " button to Save Enquiry Form.
  • Click " Close " button to Close Enquiry Form.
    • Updating of Enquiry Form
    • To Update Enquiry Form :
      • Click on " Edit " Button which is display at the top of the Form.
      • While updating the form, It is mandatory to Mention the " Reason for Updation " and Submit it.
      • Select " Utility Cost "
      • Select " Taxes On Supply "
      • Select " Taxes On Installation "
