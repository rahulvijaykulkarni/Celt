Welcome to CeltPayroll. Using this page you can enter Attendance Register Information

Attendance Register Information
  • Enter the following details To Add and View Attendance:
    • Select " Unit ".
    • Select " Full Month " or " Daily ".
    • Select " Month/year " from calendar button For example(mm/yyyy)and click"Done"or"Today".
    • Mark Attendance manually in grid view or add by using Add button.
    • Select " Date " from calendar button For example(dd/mm/yyyy).
    • Select " Status ".
  • Click " PROCESS " button to Grid view Attendance.
  • Click " ADD " button to Add Attendance.
  • Click " SAVE " button to Save Attendance.
  • Click " EXPORT TO EXCEL " button to update grid view Attendance to excel sheet .
  • Click " CLOSE " button to Close Attendance Form.
