Item Master User Manually:-

    In the Item Master you will make enter the information of instruments which provided the employee for making work in deferent location as well as for making reporting of locations.

  • Enter the following Item Master details:

    • Enter “Item Code" (System generated)
    • Select “Item Type”
      •
    • Enter “Item Name”
      •
    • Enter “Item Brand”
      •
    • Enter “Item Specification”
      •
    • Select “HSN Code”
      •
    • Select “GST Rate”
      •
    • If the GST Rate which you want not Present in the GST Rate list then click on Button For add the New GST Rate in List.
      •
    • Enter “GST Rate”
      •
    • Click Button to Save GST Rate.
      •
    • Click On Button to Close Add GST Rate Tab.
      •
    • Enter "Purchase Rate"
      •
    • Enter "Sales Rate"
      •
    • Enter "Cost/Unit"
      •
    • Enter "Validity"(In Days)
      •
    • Select "Unit"(Unit of Item)
      •
    • If the Unit which you want not Present in the Unit list then click on Button For add the New Unit in List.
      •
    • Enter “Unit Name”
      •
    • Enter "Piece per Unit"
      •
    • Click Button to Save Unit in Unit List.
      •
    • Click On Button to Close Add unit Tab.
      •
