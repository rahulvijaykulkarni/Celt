Inventory Issue


Welcome to CeltPayroll. Using this page you can Enter Inventory Issue manually.

  • Enter the following Inventory Issue Details:
    • Select " NEW " Button and Accordingly " Doc No. " will get generated.
    • Select " From Unit " in wich Inventory issue generated.
    • Select " Date " from calender button For example(dd/mm/yyyy).
    • Enter " Refrence NO. ".
    • Enter " Narration ".
    • Select " Item Name " and Accordingly " Rate " will get generated.
    • Enter " Quantity ".
    • Using Rate and Quantity Total Amount Calculated.
    • Enter " Batch No. ".
    • Enter " Pack Size ".
    • Click " + " shows the Inventory Issue Details in table.
  • Search Documents
    • Enter " Doc Number ".
    • Select " Doc Date " from calender button For example(dd/mm/yyyy).
  • Click " SEARCH " button to Search Documents Details in Enquiry Form.
  • Click " SAVE " button to Save Enquiry Form.
  • Click " UPDATE " button to Update Enquiry Form.
  • Click " DELETE " button to Delete Enquiry Form.
  • Click " CLOSE " button to Close Enquiry Form.
