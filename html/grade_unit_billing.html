Billing Rates/Client Billing


Welcome to CeltPayroll. Using this page you can enter Client Billing manually.


Client Billing User Manually:-

    Using client billing you will be make Process all types of client bill like, Manpower, Material, conveyance, deep clean by Using Attendance and Billing Rate Policy.

  • Enter the Following Detail's in Client Billing:

    • Select "Month/Year"
      •
    • Select "Client Name"
      •
    • Select "Billing Type"
      •
    • Select "State"
      •
    • Select "Branch Name"
      •
    • Select "Process Data"(by Using Current Policy/Old Policy)
      •
    • Enter "Invoice No."
      •
    • Enter "Bill Date"
      •
    • Select "Invoice Type"(Club/Unclub)
      •
    • Select "Billing Stat Day"
      •
    • Select "Billing Emd Day"
      •

    Manpower Billing:

      Using this you will be make process the manpower bill for selected client.

    • Click on Button to Process Manpower Bill for Sected Selected Mounth, Client, State and Branch.
      •
    • Click on Button to get Client Attendance for get Attendance Report for Selected Mounth, Client, State and Branch.
      •
    • Click on Button for get Rate Break up for Selected Mounth, Client, State and Branch for all Employee. The Break up Sheet will be make Prepred according to the Attendance and the Billing Policy which Defindent in the Attenced Sheet and Policy Definded in the Policy Master according to the Clientwise, Statewise, BranchWise.
      •
    • Click on Button for get Rate Break up for Selected Mounth, Client, State and Branch Designations.
    • Click on Button for get the Finance copy of Selected Mounth, Client, State and Branch for all Employee.
      •
    • Click on Button for get the Invoice copy of Selected Mounth, Client, State for all Employee.
      •
    • Click on Button for make Bill Final and get Final Invoice From the system for selected Client, Selected State and tranferd this particuler Bill Detail's to GST Report.
      •
    • Click on Button for get the Client Sheet for Manpower Bill of Selected Month.
      •
    • Click on Button for get the Report of billing according to branch will be make final or not, Branch having No Deployment, Branch Having Closed, Branch Not Approved by Admin, Branch Policy Remaning, Branch Approved by Admin, Branch not Approved by Admin, Branch Approved by finance, Branch Rejected By Finance in the selected month
    • Click on for get Status of billing for selected month and Selected Client.
      •
    • Click on Button for close the client Billing/ Billing Rates.
      •
