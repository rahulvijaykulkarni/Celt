Tripod Turnstile

Welcome to Avians. Using this page you can enter values manually.

Client Details
  • Enter the following client details:
    • Enter " Client Name ".
    • Select " Client Location ".
    • Select " Project Location ".
    • Select " Distance ".
    • Select " Product Installation Category ".
    • Enter " Project Name ".
    • Enter " Project Contact Person Name 1 ".
    • Enter " Project Contact Person No.1 ".
    • Enter " Project Contact Person Email Id 1 ".
    • Enter " Project Contact Person Name 2 ".
    • Enter " Project Contact Person No.2 ".
    • Enter " Project Contact Person Email Id 2 ".
    • Enter " Architect/Consultant Name ".
    • Enter " Architect/Consultant Contact Person No. ".
    • Enter " Architect/Consultant Contact Person Email Id ".
Technical Specifications
  • Enter the following technical details:
    • Select " Operation Type ".
    • Select " Turnstile Type ".
    • Enter " Quantity ".
    • Select " No of Lane ".
    • Select " Make ".
    • Select " Material Type ".
Additional Accessories (click on " Add More " Button)
  • If you want to Add New Accessories
    • Enter " Description ".
    • Enter " Size ".
    • Enter " Quantity ".
Advance Control Accessories
  • Enter the following Advance Control Accessories:
    • Select " Access Controller " If ' Required ' then Select " Controller Type ".
    • Enter " No. of Extra Access Cards ".
Interlocking
  • Enter the following Interlocking :
    • Enter " Avians Gate to Client System ".
    • Enter " 2 Doors ".
    • Enter " 3 Doors ".
    • Enter " 4 Doors ".
Additional Utilities (click on " Add Utilities " Button)
  • If you want to Add Extra Utilities
    • Enter " Extra Utility 1 ".
    • Enter " Extra Utility 2 ".
    • Enter " Extra Utility 3 ".
Additional Requirement
  • Select the following Additional Requirement :
    • Select " Extended Warranty ".
    • Select " Utility Cost ".
    • Select " Taxes and Duties ".
    • Select " Taxes On Installation ".
  • Click " Submit " button to Save Enquiry Form.
  • Click " Close " button to Close Enquiry Form.
    • Updating of Enquiry Form
    • To Update Enquiry Form :
      • Click on " EDIT " Button which is display at the top of the Form.
      • While updating the form, It is mandatory to Mention the " Reason for Updation " and Submit it.
      • Select " Utility Cost "
      • Select " Taxes and Duties "
      • Select " Taxes On Installation "
