Welcome to CeltPayroll. Using this page you can enter Employee Send Email manually.


Employee Send Email Manually:-

    Using this you will make send Branchwise Attendance PDF to Employee's with Attached document's. and also Send Daily Good Mornig Mail.

    • Send Email:-

      • Enter "Month and Year"
        •
      • Select "Client Name"
        •
      • Select "State Name"
        •
      • Select "Branch Name"
        •

    • Attachment:-

      • Select "Client Name"
        •
      • Select "File Name"
        •
      • Click On button to Select the document for Email.
        •
      • Click On button to Make Upload Selected Document in Mail Attechment.
        •
  • Click On button to Send Email and Send Good Morning Email.
    •
  • Click On button to Send Attendance PDF.
    •
  • Click On button to Close Employee Send Email.
