Salary Paid Report


Welcome to CeltPayroll. Using this page you can get Salary Paid Report manually.


Salary Paid Report User Manually:-

    Using Salary Paid Report you will be get information about Client Attendnce, Branch Head contact Detail's and Joinding letter sending Detail's.

  • Salary Report:

      • Here you will be get the Attendance of total employee in the selected branch for selected month.
    • Select "Month"
    • Select "Client"
    • Select "State"
      •
    • Select "Branch"
      •
    • Click on button to get Attendance excel for selected month for selected client.
      •

  • Report:

      • Using this you will be get the Branch Head detial's of the particuler branch and Report for joining letter send in the Prticuler month for selected Branch of Selected Client.
    • Select "Client"
      •
    • Select "State"
      •
    • Select "Branch"
      •
    • Select "Month"
      •
    • Select "Report Type"(Branch Head Contact Details, Joining Letter Sending Details)
      •
    • Click on button to get Joining Report Detial's or Branch Head Contact Detail.
