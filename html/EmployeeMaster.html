Employee Master User Manual:-

    In the Employee Master enter all information about the employee with their legal and Personal Details as well as mention the key responsibilities of employee in the Organization as well as Photo copy of their Documents.

    For Use Employee Master follow the following Method:

      Employee Upload:

        Using this we entered the details of employee which already in system but not completed all joining process.

      • Select “Receiver Type”(Continue, Left)

      • Click On button for Upload the Employee information in Employee Master

      • Select "Client Name"(for Sort Grid view for Client wise)
        •
      • Select "State"(for Sort Grid view for State wise)
        •
      • Select "Branch"(for Sort Grid view for Branch wise)
        •

        Use the Search Tab for Search the Particular Branch from the Grid View for making update information of Employee Master.

      • Click button to Update Employee master.
