Add Head Type Details


Welcome to CeltPayroll. Using this page you can Enter Add Head Type Details manually.

  • Enter the following Add Head Type Details:
    • Select " New Button " and Accordingly " Head Type Code " will get generated.
    • Enter " Head Type Name ".
    • Select " Head Type " .
    • Select " Head Active Status " .
    • Enter " Narration ".
  • Click On " GRID VIEW " Display Head Type Details in top of the Form.
  • Click " SAVE " button to Save Head Type Details in Form.
  • Click " UPDATE " button to Update Details in the Form.
  • Click " DELETE " button to Delete Details in the Form.
  • Click " CLOSE " button to Close the Form.
  • Click " CLEAR " button to Clear the Form.
