Purchases order


Welcome to CeltPayroll. Using this page you can enter Purchases order user manually.


Purchases order User Manually:-

    Here we Create the Purchase order before create the Purchase Bill.

  • Enter the Information in Purchase order:

    • Enter ”Purchase order No”(System Generated)
      •
    • Select “Vendor Type”
      •
    • Select “Vendor Categories”
      •
    • Select “Vendor Name”
      •
    • Get “Vendor GST No” (Get From Vendor Master)
      •
    • Enter “Purchase order Date”
      •
    • Enter “Vendor Address”
      •
    • Enter “P.O. Generated by” (Enter the name person how Generated the P.O.)
      •
    • Enter “Mobile No”
      •
    • Enter “Advance in %” (In Percentages)
      •
    • Enter “Remaining % After Advance”
      •

  • Add Item:

    • Select “Item Type”
      •
    • Select “Item Name”
      •
    • Enter “Description” (Enter description on this item)
      •
    • Enter “HSN Code” (HSN Code of Item)
      •
    • Enter “GST Rate”
      •
    • Enter “Quantity” (of Item)
      •
    • Enter “Rate per Unit”
      •
    • Then Get "Gross Bill Amount of Item"
      •  Click On button to Add Item in Bill.

  • Then You will Get the Calculation of Taxes:

    • Sub Total of “Total Bill”
      •
    • Amount of CGST
      •
    • Amount of SGST
      •
    • If IGST then get Amount of IGST
      •
    • Grand Total(After Adding GST Amount)
      •
    • Advance Payment(IF Make Advance Payment)
      •
    • Remaining Payment(After Deducting Advance Payment Amount)
      •
  • Click On button to Purchase Order.
    •
  • Click On button to Get Purchase Order Invoice.
    •

    • For Search:

      • Enter “Purchase Order No” Or
        •
      • Enter “Vendor Name”
        •
      • Then Click on button to Search Purchase order.
        •
      • Then Click on button and make changes you want.
        •
  • If you want make any changes in Purchase order then make it.
    •
  • Click On button to Update information in Purchase Order.
    •
  • Click On button to Delete Purchase Order.
    •
  • Click On button to Make Purchase Order Final.
    •
  • Click On button to Close Purchase Order.
    •
