Inventory transaction


Welcome to CeltPayroll. Using this page you can enter Inventory transaction information.

  • Enter the following Inventory transaction details:
    • Select " From Date " from calendar button For example(dd/mm/yyyy).
    • Select " To Date " from calendar button For example(dd/mm/yyyy).
  • Click " VIEW DATA " Button to display Inventory transaction.
  • Click " EXPORT TO EXCEL " to see execl sheet.
  • Click " CLOSE " Button to close the Inventory transaction form.
