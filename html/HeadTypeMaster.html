Head Type Master


Welcome to CeltPayroll. Using this page you can enter Head Type Master information.

  • Enter the following Head Type Master details:
    • " Head Type Code " will be auto genrated.
    • Enter " Head Type Name ".
    • Select " Head Type ".
    • Select " Active status ".
  • Click " SAVE " Button to save HeadTypeMaster.
  • Click " CLEAR " Button to clear the HeadTypeMaster.
  • Click " EXPORT TO EXCEL " to see excel sheet.
  • Click " CLOSE " Button to close the HeadTypeMaster form.
  • Click " -> " arrow to update the record.
  • Click " UPDATE " button to save updated record.
  • Click " DELETE " button to delete the record.
