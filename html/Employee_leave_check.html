Employee Leave Detail's User Manually:-

    Using Employee Leave Detail's get the information about leave application of employee which they will make submmited by using mobile application and from here admin will be make this Approve or Reject.

  • Enter the following details for Employee Leave Detail's:

    • Select "Client Name"
      •
    • Select "State"
      •
    • Select "Unit Name"
      •
    • Enter "From Date"
      •
    • Enter "To Date"
      •
    • Click On Button to Show the employee Detail's in Grid View.
      •
    • Click On Button to Approve Leave.
      •
    • Click On Button to Reject Leave.
      •
    • Click On Button to close Employee Leave Detail's.
