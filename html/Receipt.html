Sundry Receipt Details


Welcome to CeltPayroll. Using this page you can Enter Employee Receipt Details manually.

  • Enter the following Receipt Details:
    • Select " NEW " Button and Accordingly " CashSet " will get generated.
    • Select " Date " from calendar button For example(dd/mm/yyyy).
    • Select " Bank " .
    • Select " Head Code " .
    • Enter " Employee Code ".
    • Enter " Employee Name ".
    • Enter " Amount ".
    • Enter " Narration ".
    • Click On " + " shows the Receipt Details in Table.
    • Click On " GRID VIEW " Delete Details in Table.
  • Searching Employee Details
    • Enter " CashSet Number ".
    • Select " CashSet Date " from calendar button For example(dd/mm/yyyy).
  • Click " SEARCH EMP " button to Search Employee Details in receipt Form.
  • Click " SAVE " button to Save Receipt Details in receipt Form.
  • Click " UPDATE " button to Update receipt Form.
  • Click " DELETE " button to Delete receipt Form.
  • Click " CLOSE " button to Close receipt Form.
