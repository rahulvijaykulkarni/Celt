Advance Policy


Welcome to CeltPayroll. Using this page you can enter Advance Policy manually.


Advance Policy User Manually:-

    Using this we make the advance payemet to employee as well as vendor.

  • Employee Advance:

      • Using this we make the Advance payment to employee.
    • Select "Client Name"
      •
    • Select "State"
      •
    • Select "Branch Name"
      •
    • Select "User Type"(Employee/ Field Officer)
      •
    • Select "Advance For Services"
      •
    • Select "Employee Name"
      •
    • Select "Field Officer Name"
      •
    • Enter "Amount"
      •
    • Enter "Date"
      •
    • Enter "Description" of Pyament
      •
    • Select "Payment Mode"
      •
    • Click On Button to Save the payment record in Advance Policy.
      •
    • Click On Button to get the report of Advance Payment.
      •

  • Vendor Advance:

      • Using this we make the Advance payment to Vendor.
    • Select "Vendor Name"
      •
    • Select "PO No"
      •
    • Select "Invoice No"
      •
    • Enter "Amount"
      •
    • Enter "Date"
      •
    • Enter "Description" of Pyament
      •
    • Select "Payment Mode"(cheque, Online)
      •
    • If Cheque then Enter "Cheque No."
      •
    • Enter "Cheque Received By"
      •
    • If Online then Enter "UTR No."
      •
    • Click On Button to Save the payment record in Advance Policy.
      •
    • Click On Button to Record from Advance Payment.
      •
    • Click On Button to Close Advance Payment.
      •
