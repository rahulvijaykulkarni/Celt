Sundary Payment Details


Welcome to CeltPayroll. Using this page you can enter Cheque(AP) information.

  • Enter the following Cheque(AP) details:
    • Select " NEW " button " cashset code" will auto generated .
    • Select " Date " from calendar button For example(dd/mm/yyyy).
    • Select " Bank ".
    • Select " Head code ".
    • Select " Customer ".
    • Enter " Employee Name ".
    • Enter " Employee Code ".
    • Enter " Narration ".
    • Enter " Cheque no.".
    • Select " Cheque Date " from calendar button For example(dd/mm/yyyy).
    • Enter " Bank details "
    • Enter " Amount "
    • Enter " TDS "
    • Enter "Net Amount "
    • Enter " Cashset Number "
    • Select " Cashset Date " from calendar button For example(dd/mm/yyyy).
  • Click " + " Button to display Cheque(AR) records
  • Click " SAVE " Button to save Cheque(AR) records.
  • Click "SEARCH " to Search record in Cheque(AR)
  • Click " GET DETAILS " to get details of record in table
  • Click " UPDATE " Button to updated Cheque(AR) records.
  • Click " DELETE " Button to Delete Cheque(AR) records
  • Click " CLOSE " Button to close the Cheque(AR) form.
